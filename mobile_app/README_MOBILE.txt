LE MEILLEUR QUIZ DE H.M - VERSION MOBILE

Cette version fonctionne sur telephone ET PC via navigateur.

Fichier principal:
- mobile_app/index.html

Utilisation rapide (PC):
1) Ouvrir mobile_app/index.html dans Chrome/Edge
2) Choisir un domaine
3) Cliquer sur "Tu es pret ? Commencer"

Utilisation sur telephone:
1) Envoyer le fichier index.html (ou le dossier mobile_app) vers ton telephone
2) Ouvrir index.html dans le navigateur
3) Jouer

Option recommandee:
- Heberger le dossier mobile_app sur GitHub Pages / Netlify pour avoir un lien web.
